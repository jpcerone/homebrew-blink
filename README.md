# Homebrew Tap for Blink Launcher

A fast application launcher for macOS.

## Installation

```bash
brew tap jpcerone/blink
brew install --cask blink-launcher
```

## Updating

```bash
brew upgrade --cask blink-launcher
```

## Uninstalling

```bash
brew uninstall --cask blink-launcher
```

This will also remove the config directory at `~/.config/blink`.
